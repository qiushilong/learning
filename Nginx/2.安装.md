# 安装

## debian

```bash
# 更新包索引
sudo apt update

# 安装 nginx 软件包
sudo apt install nginx

# 测试安装
curl -I 127.0.0.1

# 停止 nginx 服务
sudo systemctl stop nginx

# 启动 nginx 服务
sudo systemctl start nginx

# 重启 nginx 服务
sudo systemctl restart nginx

# 配置更改后重新加载 nginx 服务
sudo systemctl reload nginx

# 禁用 nginx 服务在系统启动时启动
sudo systemctl disable nginx

# 启用 nginx 服务在系统启动时启动
sudo systemctl enable nginx
```
